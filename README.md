# coffeetracker

Demo App bult with Aurelia

1. Ensure that [NodeJS](http://nodejs.org/), [Gulp](http://gulpjs.com/) and [jspm](http://jspm.io/) are installed globally.

  ```shell
  npm install
  ```
  
    ```shell
  npm install -g gulp
  ```
  
    ```shell
  npm install -g jspm
  ```
  
    ```shell
  jspm install
  ```
  
2. To run the app, run the command:
  
    ```shell
  gulp watch
  ```
